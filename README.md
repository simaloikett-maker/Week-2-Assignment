# Week-2-Assignment
# Personal Budget Tracker

## Project Overview

This project is a simple Personal Budget Tracker created using HTML and CSS.

It allows users to:

- Enter expense information using a form.
- View sample expenses in a structured table.
- Read instructions using a collapsible details section.
- Watch a budgeting tips video.
- Experience improved styling using advanced CSS selectors.

---

## Project Files

### index.html

Contains the structure of the webpage including:

- Header
- Logo image
- Expense form
- Expense table
- Details and Summary section
- Embedded YouTube video

---

### style.css

Contains all page styling including:

- Layout styling
- Form styling
- Button styling
- Table styling
- Hover effects
- Alternating table rows
- Advanced CSS selectors
